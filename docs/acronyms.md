# Acronyms

This glossary defines the abbreviations used across `PyPNM-CMTS-WebUI` docs,
code, and UI labels.

| Acronym | Meaning |
| --- | --- |
| API | Application Programming Interface |
| CA | Coding Agent guidance (`CODING_AGENTS.md`) |
| CI | Continuous Integration |
| CM | Cable Modem |
| CMTS | Cable Modem Termination System |
| CW | Codeword |
| FEC | Forward Error Correction |
| OFDM | Orthogonal Frequency Division Multiplexing |
| OFDMA | Orthogonal Frequency Division Multiple Access |
| PCW | PyPNM-CMTS-WebUI project shorthand |
| PNM | Proactive Network Maintenance |
| PW | PyPNM-WebUI project shorthand |
| RxMER | Receive Modulation Error Ratio |
| SA | Spectrum Analyzer |
| SCQAM | Single-Carrier Quadrature Amplitude Modulation |
| SG | Serving Group |
| SNMP | Simple Network Management Protocol |
| TFTP | Trivial File Transfer Protocol |
| UI | User Interface |
| UX | User Experience |
